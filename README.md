# DINO-PAC 🦖

A prehistoric Pac-Man built with vanilla JavaScript and HTML5 Canvas. No dependencies, no build step.

**[▶ Play now](https://vkazei.github.io/dinopac/)**
*Or add to home screen on Android/iOS for a fullscreen app experience.*

---

## How to play

| Input | Action |
|---|---|
| Arrow keys / WASD | Move |
| Swipe | Move (mobile) |
| On-screen D-pad | Move (touch) |

- Eat all the green eggs to advance to the next level
- Big eggs give you temporary power — eat the pterodactyls while powered!
- Grab the 🦴 fossil bonus when it appears

## Run locally

```
python3 -m http.server
```
Then open `http://localhost:8000` — no install needed.

## Tech

- Vanilla JS + HTML5 Canvas 2D
- Web Audio API (procedural sound effects)
- PWA — installable, works offline
- Zero dependencies, zero build tools
